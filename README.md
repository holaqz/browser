## Структура проекта

```
browser/
├── index.html          # HTML-шаблон приложения
├── main.js             # Основной процесс Electron
├── preload.js          # Предзагрузочный скрипт для контекста
├── renderer.js         # Рендеринг процесса
├── browser_api.js      # API для работы с событиями в браузере
├── ipc_handlers.js     # Обработчики IPC событий
├── package.json        # Зависимости и настройки проекта
├── package-lock.json   # Блокировка версий зависимостей
```

## Установка и запуск

### Требования
- Node.js (версия 14 или выше)
- npm или yarn

### Установка зависимостей
```bash
npm install
```

### Запуск приложения
```bash
npm start
```

## API для разработчиков

После запуска приложения в консоли браузера доступен объект `window.Browser` с следующими методами:

- `.enableTracking()` - включить отслеживание
- `.disableTracking()` - выключить отслеживание
- `.getTrackingState()` - получить текущее состояние отслеживания
- `.getSessionEvents()` - получить все события сессии
- `.getEventsLastMinutes(minutes)` - получить события за последние N минут
- `.getStats()` - получить статистику событий
